The CSS way

There are 3 ways to use CSS

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS

                <button style="color:brown">Click Me!</button>

Internal CSS

In the head section of the HTML document as below:

                <style>
                    button {
                            color: brown;
                        }
                </style>

And then you can have a button like:

                <button>Click Me!</button>

External CSS

In another file

                button {
                        color: brown;
                    }

And it is imported into the HTML using a link tag

                <link rel="stylesheet" href="main.css">
